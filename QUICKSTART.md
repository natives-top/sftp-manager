# 🚀 快速开始指南

## Windows 版本 (v1.3.0)

### 📥 获取应用

**本地构建产物**:
```
D:\sftp-tools-ui\sftp-manager\release\现代化 SFTP 文件管理-1.3.0.exe
```

**文件大小**: 169 MB  
**类型**: Windows PE32+ 可执行文件 (x86-64)

### ▶️ 启动应用

1. **直接双击** exe 文件即可运行
2. **无需安装** - 完全便携式
3. **无需依赖** - 包含 Electron 运行时

### 🔧 首次使用

#### 第一步: 创建连接
1. 点击 "新建连接"
2. 填写以下信息:
   - **主机 IP**: SFTP 服务器地址
   - **端口**: 通常为 22
   - **用户名**: SFTP 用户名
   - **密码**: SFTP 密码
   - **名称** (可选): 连接别名，便于识别

#### 第二步: 连接到服务器
1. 点击 "连接" 按钮
2. 等待连接建立
3. 连接成功后会显示远程文件列表

#### 第三步: 开始文件管理
- **上传**: 选择文件 → 点击上传
- **下载**: 右键文件 → 下载
- **删除**: 右键文件 → 删除
- **新建文件夹**: 点击新建文件夹

### 📌 使用书签和别名

#### 创建书签
1. 右键点击任何文件或文件夹
2. 选择 "标签" (Bookmark)
3. 在对话框中输入 **别名** (可选)
4. 点击 "保存"

#### 使用书签
- 书签出现在左侧栏 "标签列表" 下
- 点击书签快速导航到该位置
- **有别名**: 显示别名 + 文件名 (灰色)
- **无别名**: 显示原始文件名

#### 编辑别名
1. 在左侧栏右键书签
2. 选择 "编辑别名"
3. 修改或删除别名
4. 点击 "保存"

### 🌐 切换语言

点击左下角的语言切换按钮:
- **English** - 英文界面
- **中文** - 中文界面

### ⚙️ 常见问题

#### Q: 如何添加多个连接?
A: 每个连接独立管理。创建新连接后会显示在左侧栏中。点击连接名即可切换。

#### Q: 文件传输时能取消吗?
A: 可以。在 "传输任务" 面板中找到任务并点击删除按钮。

#### Q: 密码会被保存吗?
A: 会被加密保存到本地。密码使用 AES-256-GCM 加密。

#### Q: 支持哪些文件操作?
A: 支持上传、下载、删除、创建文件夹、查看文件属性等。

#### Q: 可以同时传输多个文件吗?
A: 可以。任务会排队执行。

### 📱 界面说明

#### 左侧栏
- **连接列表** - 已保存的 SFTP 连接
- **书签** - 快速访问的路径

- 使用标准 SFTP 协议
