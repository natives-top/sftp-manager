## 技术实现细节

### 状态管理
```typescript
// 存储选中的文件索引集合 (O(1) 查询性能)
const [selectedFileIndices, setSelectedFileIndices] = useState<Set<number>>(new Set())

// 记录上一次选中的索引 (用于 Shift 范围选择)
const [lastSelectedIndex, setLastSelectedIndex] = useState<number | null>(null)
```

### 选择逻辑
- 使用 `Set<number>` 存储选中文件的索引
- 单次/多次/范围选择的逻辑清晰独立
- 性能高效,支持大量文件

### 批量下载流程
1. 获取所有选中的文件
2. 按索引排序
3. 遍历每个文件调用 `handleDownload()`
4. 每个文件创建独立的 TransferTask
5. 下载完成后自动清除选择

## 用户工作流

### 典型使用场景

**场景 1: 下载特定文件**
```
1. 点击复选框或 Ctrl+单击选中多个文件
2. 工具栏显示 "Download (3)"
3. 点击按钮或右键菜单选择下载
4. 三个文件同时开始传输
```

**场景 2: 下载一个范围**
```
1. 点击第一个文件
2. Shift+单击最后一个文件
3. 中间所有文件自动选中
4. 点击 "Download" 开始批量下载
```

**场景 3: 全部下载**
```
1. 点击表头复选框全选所有文件
2. 点击 "Download (全部数量)" 按钮
3. 所有文件同时开始下载
```
