# 现代化 SFTP 文件管理 - 使用指南

## 快速开始

### Windows 用户

1. **下载应用**
   - 从 GitHub Releases 下载 `现代化 SFTP 文件管理-win32-x64` 文件夹

2. **运行应用**
   - **方法 1 (推荐)**: 在 release 目录中双击 `launch-app.bat`
   - **方法 2**: 打开 `现代化 SFTP 文件管理-win32-x64` 文件夹，双击 `现代化 SFTP 文件管理.exe`

3. **首次使用**
   - 应用会启动，显示连接界面
   - 点击 "新建连接" 添加 SFTP 服务器

## 系统要求

- **操作系统**: Windows 7 或更高版本 (推荐 Windows 10/11)
- **磁盘空间**: 至少 250MB
- **内存**: 最少 256MB RAM
- **网络**: 用于 SFTP 连接的互联网访问

## 连接设置

### 基本连接参数
```
主机名: SFTP 服务器地址 (例: sftp.example.com)
端口: 22 (通常)
用户名: 你的 SFTP 用户名
密码: 或者密钥认证
```

### 高级选项
- **密钥认证**: 上传 private key 文件
- **连接超时**: 自定义连接超时时间
- **SSH 配置**: 支持自定义 SSH 参数

## 常见问题

### Q: 无法连接到 SFTP 服务器
**A**: 
- 检查服务器地址和端口是否正确
- 验证用户名和密码
- 确保防火墙允许 SSH 连接 (端口 22)
- 检查网络连接

### Q: 文件传输很慢
**A**:
- 这取决于你的网络连接速度
- 检查服务器负载
- 减少同时传输的文件数量

### Q: 应用提示 "ffmpeg.dll 缺失"
**A**:
- 这是极少见的 Windows 系统库问题
- 尝试安装最新的 Visual C++ Redistributable
- 检查 Windows 更新

### Q: 如何连接使用密钥认证?
**A**:
- 在连接设置中选择"密钥认证"
- 上传你的 private key 文件 (通常是 `id_rsa`)
- 如果密钥有密码,输入密码

## 数据隐私

- 所有连接信息保存在本地 (`~/.config/electron-app/storage.json`)
- 密码被加密存储
- 无任何数据上传到云端
- 完全离线可用
